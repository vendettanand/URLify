Requirement:

1. MongoDB 3.6
2. Apache Tomcat 8.5
3. Java 1.8

Instructions:
1. Install MongoDB and start it.
2. Create a user on MongoDB with credentials: "user" and "password".
3. Create a database on MongoDB with name: "db"
2. Copy the URLify.war to the webapps folder of Apache Tomcat and start the server.
3. Open browser and enter the URL: "http://localhost:8080/URLify"
